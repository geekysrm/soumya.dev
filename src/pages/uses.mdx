### Browser Extensions / Addons

- [uBlock Origin](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/) - Trusty ad-blocker which does its job perfectly.
- [Nano Defender](https://addons.mozilla.org/en-US/firefox/addon/nano-defender-firefox/) - Anti-adblock defuser.
- [Privacy Badger](https://addons.mozilla.org/en-US/firefox/addon/privacy-badger17/) - Automatic Tracker blocking and link tracking.
- [Bitwarden Password Manager](https://addons.mozilla.org/en-US/firefox/addon/bitwarden-password-manager/) - Used to store and sync passwords across devices.
- [DF Youtube](https://addons.mozilla.org/en-US/firefox/addon/df-youtube/) - Makes youtube distraction free.
- [Feedbro](https://addons.mozilla.org/en-US/firefox/addon/feedbroreader/) -The best RSS Feed Reader.
- [LeechBlock NG](https://addons.mozilla.org/en-US/firefox/addon/leechblock-ng/) - Productivity tool which blocks certain websites for a time period.
- [React Developer Tools](https://addons.mozilla.org/en-US/firefox/addon/react-devtools/) - Used to debug React apps.
- [m-wiki](https://addons.mozilla.org/en-US/firefox/addon/m-wiki/) - Redirect desktop Wikipedia to the cleaner mobile version.
- [Minimal Twitter](https://addons.mozilla.org/en-US/firefox/addon/min-twitter/) - Make twitter minimal and pleasant to use.
- [Neat URL](https://addons.mozilla.org/en-US/firefox/addon/neat-url/) - Remove all tracking parameters from URLs.
- [Octotree](https://addons.mozilla.org/en-US/firefox/addon/octotree/) - Adds a file structure explorer to GitHub.
- [Refined GitHub](https://addons.mozilla.org/en-US/firefox/addon/refined-github-/) - Adds many useful features to GitHub.
- [Session Boss](https://addons.mozilla.org/en-US/firefox/addon/session-boss/) - Useful for saving browser sessions for later use.
- [Simplify Gmail](https://addons.mozilla.org/en-US/firefox/addon/simplifygmail/) - Removes clutter and simplifies the UI of Gmail web.
- [Stylus](https://addons.mozilla.org/en-US/firefox/addon/styl-us/) - Used to add custom styles to any webpage.
- [Tampermonkey](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/) - Used to add userscripts (custom javascript) to any webpage.
- [Video Speed Controller](https://addons.mozilla.org/en-US/firefox/addon/videospeed/) - Adds a speed controller to video elements.
- [Vimium](https://addons.mozilla.org/en-US/firefox/addon/vimium-ff/) - Provides keyboard shortcuts for navigation and control like Vim.
- [nightTab](https://addons.mozilla.org/en-US/firefox/addon/nighttab/) - A clean, customizable new tab extension.
- [UnBlur Quora](https://addons.mozilla.org/en-US/firefox/addon/unblur-quora/) (developed by me)- Remove annoying signup popups while browsing Quora.
